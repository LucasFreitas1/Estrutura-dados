\question[2] Construa um algoritmo (função) que coloque em um vetor os 10 primeiros valores da seqüência de Fibonacci. A seqüência de Fibonacci é definida pela seguinte fórmula:

\[
  f(n) = \left\{ 
  \begin{array}{l l}
    1 & se \ n = 0  \\
    1 & se \ n = 1 \\
    f(n - 1) + f(n - 2) & se \ n \  >= 1 \\
  \end{array} \right.
\]

\question[2] Construa um algoritmo (função) que coloque em um vetor os 10 primeiros  valores da função fatorial. Por exemplo, o elemento v[0] deve conter o fatorial de 0, o elemento v[1] o fatorial de 1 e assim por diante. Para calcular o valor do  fatorial de cada número é útil empregar a seguinte definição.

\[
  fatorian(n) = \left\{ 
  \begin{array}{l l}
    1 & se \ n = 0  \\
    n * fatorial (n - 1) & se \ n > \ 0\\
  \end{array} \right.
\]
