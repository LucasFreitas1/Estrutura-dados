\large \def\ExerciseName{Lista de Exercícios 3}

\begin{Exercise*}

\Question Construa uma função que retorne o tamanho de uma lista estática.

\Question Dada uma lista estática de tamanho $m$, construa uma função que inverta a lista.

\Question Dada uma lista estática de tamanho $m$, construir uma função que retorne uma \textit{sublista} de tamanho $n$ a partir da posição $p$. Compute a complexidade de tempo da sua função.

\Question Dada duas listas estáticas de tamanho $m_1$ e $m_2$, respectivamente, construir uma função para intercalar as duas listas, gerando uma terceira.

\Question Dada duas listas estáticas de tamanho $m_1$ e $m_2$, respectivamente, construir uma função para concatenar as duas listas, gerando uma terceira.

\Question Uma maneira usual de representar conjuntos é listando seus elementos. Implemente uma aplicação que ofereça as operações usuais de conjuntos (união, intersecção e diferença), considerando que cada um dos conjuntos é representado por uma lista linear.
		
\end{Exercise*}
