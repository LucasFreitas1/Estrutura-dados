\Question A seguinte função calcula o Máximo Divisor Comum (MDC) dos inteiros positivos \textit{m} e \textit{n}. Escreva uma função recursiva equivalente.

\begin{lstlisting}
int mdc(int m, int n){
 int r;
 do{
   r = m % n;
   m = n; n = r;
 }while(r != 0); 
 return m;
}
\end{lstlisting}
